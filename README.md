# PDFx – Merge PDFs

A web application built with Node.js and Express.js that allows users to upload and merge multiple PDF files into a single PDF document.

## 🚀 Live Demo

https://pdfx-merge-pdfs.onrender.com

## ✨ Features

- Merge multiple PDFs
- Fast processing
- Secure uploads
- No watermark
- Modern UI

## 🛠️ Tech Stack

- Node.js
- Express.js
- HTML
- CSS
- JavaScript

## ▶️ Run Locally

npm install
npm start
