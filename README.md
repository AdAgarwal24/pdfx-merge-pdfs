# PDFx – Merge PDFs

A Node.js web application to merge multiple PDF files instantly.

## Live Demo
https://pdfx-merge-pdfs.onrender.com

## Features
- Merge multiple PDFs
- Fast processing
- Secure uploads
- No watermark
- Modern UI

## Tech Stack
- Node.js
- Express.js
- HTML
- CSS
- JavaScript

## Run Locally

```bash
npm install
npm start
